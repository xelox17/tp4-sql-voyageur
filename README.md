# TP4 – SQL (Base Voyageur)

Ce dépôt contient les réponses au TP4 de SQL portant sur :
- les sélections et filtres
- les jointures (INNER, LEFT, RIGHT)
- les sous-requêtes
- les fonctions d’agrégation
- les opérations ensemblistes

## Organisation du dépôt
- `tp4.sql` : contient l’ensemble des requêtes SQL répondant aux questions du TP.
- `README.md` : fichier de présentation du projet.

## Exécution
Les requêtes peuvent être exécutées dans le SGBD utilisé en cours
(MySQL / PostgreSQL / SQLite selon le cas).

Chaque requête est commentée dans le fichier `tp4.sql`
afin d’indiquer clairement à quelle question elle correspond.
